Daily Quest tracker for CatsEyeXI Private Server. Parses chat log for quest dialogue and stores relevant information in a UI table for easy readability.

//lua l dailytask
Accepts //dailytask and //dtask

//dailytask show		        --Show the UI
//dailytask hide		        --Hide the UI
//dailytask toggle		      --Toggle show/hide UI
//dailytask clear | reset	  --Clear the UI table
//dailytask test		        --Display test information
//dailytask pos x y		      --Fine tune position
//dailytask font size		    --Adjust font size



UI table is draggable by default but position command is added just in case it's lost somewhere outside a smaller resolution window.
